MapData Diff Viewer

Generate diff GeoJSON with: 
python3 cabot-navigation/mf_localization_mapping/script/diff_mapdata_geojson.py --orig ... --shifted ... -o /tmp/MapData.diff.geojson --delta_lines --summary
orig shifted delta 
Load a GeoJSON diff to see counts.
Mouse: drag to pan, wheel to zoom. This viewer uses a simple lon/lat plane (no basemap).
